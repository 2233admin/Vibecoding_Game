# Gameplay Loop

Use this file when rebuilding or reviewing the demo's actual player journey.

## Target Loop

The current demo should feel complete across this path:
- start a new game
- receive a clear first objective
- walk the first route without confusion
- encounter and capture at least one monster
- understand healing, switching, and item use
- prepare for the first gym
- enter and beat the first gym
- receive a reward and continue with save/load intact

## Loop Principles

- clarity before quantity
- strategy before raw inflation
- momentum before world scale

## Opening Segment

The player should quickly understand:
- who they are
- where to go next
- why the first route matters

If the player can move but does not know what to do, the opening is not solved.

## Capture Segment

The player should:
- trigger a wild encounter early
- understand capture as a core verb
- feel that collecting partners matters to the later gym challenge

The first capture should reinforce momentum, not interrupt it with too many side systems.

## Growth Segment

Before the first gym, the player should naturally learn:
- healing
- switching
- item usage
- move limit and move choice
- basic type advantage thinking

This segment should teach through play, not only through text.

## Gym Segment

The gym should test whether the player has understood:
- team building
- switching decisions
- item timing
- attribute matchup awareness

The gym should feel winnable with thought, not trivial and not grind-gated.

## Reward Segment

After gym victory, the player should see:
- clear progression reward
- updated flags or unlocked next intent
- continuity with the larger authority/trial theme

Important:
- badge victory proves growth
- it does not instantly grant divine-scale payoff

## Rebuild Focus

When rebuilding the loop, prioritize in this order:
1. opening clarity
2. capture reliability
3. battle readability
4. gym access clarity
5. win/reward/save continuity

If a feature does not improve one of those five, it is probably not first-priority.

这个文件用于新开会话时，快速让新的 agent 进入正确上下文。

```text

Goal: continue rebuilding the web demo so the first-gym loop is clear and stable.
```

## 适用场景

用简短版：
- 会话很轻
- 你只想让 agent 先正确加载项目上下文

用强化版：
- 代码已经有些乱
- 你要 agent 直接进入 demo 恢复 / 重建状态
- 你希望它优先收口首个道馆闭环

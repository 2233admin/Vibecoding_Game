# 蒙德草原版本完整规划（执行版）

## 1. 版本目标

在保持现有 Web 架构不重构的前提下，交付“新手村 -> 蒙德草原 -> 草系道馆”的完整可玩闭环，并将传说遭遇、草系机制教学、吞噬/融合实操接入到主线体验中。

核心体验目标：
- 教学 -> 实践 -> 受挫 -> 引导 -> 掌握

## 2. 体验主线（锁定）

### 阶段 A：新手村（选择与代入）
1. 选择初始精灵（草/火/水），触发首次剧情战（无惩罚）。
2. 领取图鉴与基础道具，明确前往蒙德草原与草系道馆目标。
3. 至少一次“吞噬/融合机制介绍”对话（概念教学，不强制执行）。

### 阶段 B：蒙德草原入口（基础实践）
1. 完成野外遇敌与捕捉教学（NPC 示范一次）。
2. 轻量草系机制首次实操（续航/控制核心：吸取、寄生、麻痹/睡眠）。
3. 主线增加一次门槛：进入中段前，至少完成一次吞噬或融合（二选一）。

### 阶段 C：蒙德草原中段（主线冲突）
1. 反派组织首次登场并进行第 1 战。
2. 战后给出草系道馆与神兽线索。

### 阶段 D：蒙德草原后段/道馆前（难度抬升）
1. 劲敌战或训练师连战，检验队伍构筑。
2. 反派第 2 战（道馆前拦截），通过后开放道馆挑战。

### 阶段 E：草系道馆（闭环收口）
1. 道馆机制强调属性克制与队伍搭配。
2. 道馆前提供对策资源，避免卡关。
3. 通关后给主线线索并进入流派/神兽后续分支。

## 3. 精灵与属性扩展（锁定）

### 3.1 蒙德草原新增 4 只稀有精灵
命名规则：避免“王/古/圣裁”等传说感词汇，和神兽层拉开。

当前命名方向（可在实现前最后确认）：
1. 岚叶灵鹿
2. 雾孢行者
3. 风棘隼
4. 藤角犀

### 3.2 冰属性接入
定位：高输出 + 强控制，但防御弱。

克制规则：
- 进攻：冰 -> 草/飞行/地面 = x2
- 防守弱点：火/格斗/岩石 = x2
- 抗性：冰 = x0.5

## 4. 传说精灵机制（锁定）

1. 道馆前最多遭遇/交手 3 只传说精灵。
2. 道馆前允许捕捉，但为低概率档（基础 0.8%，条件加成后上限 <= 3%）。
3. 道馆前玩家至少遇到 1 只传说的概率 >= 80%：
   - 采用“基础触发 + 保底递增”。
4. 当玩家击败草系道馆，或获得草系神兽权能后：
   - 森律神使捕捉率提升（建议基础 4%，条件达成后上限 8-10%）。
5. 口径统一：
   - 曜辉圣兽与炽脉兽视作同一传说位（同一物种，不重复开图鉴位）。

## 5. 实施顺序（从现在开始）

### P0（立即）
1. 写入版本规则到规划与看板（已完成）。
2. 接入“新手村介绍一次吞噬/融合”。
3. 接入“主线至少完成一次吞噬/融合”门槛。

### P1
1. 冰属性完整接线（type 表、克制表、UI 文案、技能）。
2. 草系前期技能生态重配（续航+控制优先）。

### P2
1. 蒙德草原 4 只稀有精灵落地（物种、遭遇、技能、图鉴文案）。
2. 传说遭遇判定 + 80% 保障机制落地。

### P3
1. 反派两战接线与关卡门槛收束。
2. 草系道馆前后节奏与奖励校准。

### P4
1. 回归验收脚本扩展到新链路。
2. 文档/知识库同步，出可演示脚本。

## 6. 验收标准

1. 玩家在新手村可明确理解吞噬/融合差异。
2. 玩家在道馆前主线中至少实际执行一次吞噬或融合。
3. 冰属性克制规则生效且不破坏现有主循环。
4. 蒙德草原新增 4 只稀有精灵可稳定遭遇并有清晰战术定位。
5. 道馆前传说遭遇符合“最多 3 只 + 至少 80% 遇见 1 只”目标。
6. 道馆通关/草系权能后森律神使捕捉率提升生效。

## 7. 风险与策略

1. 风险：道馆前内容过满导致节奏拖沓。  
   策略：主线只保留必要节点，支线奖励化。

2. 风险：传说遭遇过频影响平衡。  
   策略：触发窗口 + 冷却 + 上限次数。

3. 风险：机制教学打断沉浸。  
   策略：新手村仅概念引导，实操放草原入口并配奖励。

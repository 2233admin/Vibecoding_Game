# 特性总表 v1（B/A/S 分级）

生成日期：2026-03-28
来源文档：`docs/balance-agent-traits-requirements-v1.md`、`docs/trait-tier-balance-spec.md`
配置文件：`config/trait-balance.json`

---

## 一、分级总览

| 级别 | 定位 | 强度上限 | 首道馆前可投放 | 典型持有者 |
|------|------|----------|--------------|-----------|
| **B** | 基础增益，提升手感，不改变玩法范式 | 单项 +5%~15% | ✅ | 稀有精灵 |
| **A** | 流派核心，明显影响战斗/探索决策，有触发边界 | 有条件激活 | ✅ | 传说精灵 / 神兽 |
| **S** | 后期稀缺，强力上限 | 高，无受限区间 | ❌ 禁止 | 神兽 / 特殊进化 |

**投放硬约束：**
- 普通精灵：无特性
- 稀有/传说/神兽：可配置 B/A/S 特性
- 首道馆前：`tierPolicy.preGymAllowSTier = false`，S 级一律不发放
- 同类效果不叠加，取最高级别值（`stackingPolicy.priority = [S, A, B]`）
- 冒险特性（`adventureActiveInBag: true`）：持有者在背包/后备即可生效

## 二、特性条目详表

### B 级特性

#### `blaze_boost_b` — 火势增幅

| 项目 | 值 |
|------|----|
| 类型 | battle（战斗） |
| 效果 | 持有者使用火系技能伤害 **+10%** |
| 触发条件 | 技能属性 = `fire` |
| 建议区间 | +5% ~ +15% |
| 当前取值 | +10%（中位） |
| 背包生效 | ❌（出战才生效） |
| 叠加组 | `type_boost`（同组不叠加） |
| 首道馆前 | ✅ 允许 |
| 风险备注 | 叠加属性相性（STAB + 克制）最高可达 10%×1.5×2 = 实际倍率较高，建议配合属性平衡监控 |

---

#### `fervor_b` — 奋勇

| 项目 | 值 |
|------|----|
| 类型 | battle（战斗） |
| 效果 | HP < 50% 时，按缺血比例线性增伤，**上限 +15%** |
| 触发条件 | 当前 HP < maxHp × 50% |
| 增伤公式 | `bonus = 0.15 × (1 - currentHp / (maxHp × 0.5))`，clamp(0, 0.15) |
| 建议区间 | 上限 +8% ~ +20% |
| 当前取值 | 上限 +15% |
| 背包生效 | ❌（出战才生效） |
| 叠加组 | `low_hp_boost` |
| 首道馆前 | ✅ 允许 |
| 风险备注 | 配合 `unyielding_s（不屈）`可造成"保 1 HP + 全额奋勇增伤"的组合，首道馆前禁止同时持有 S 级可规避 |

---

### A 级特性

#### `fearless_stride_a` — 一往无前

| 项目 | 值 |
|------|----|
| 类型 | adventure（冒险） |
| 效果 | 对等级**严格低于**持有者等级的野生精灵，逃跑成功率 100% |
| 触发条件 | 持有者等级 > 目标野怪等级 **且** 目标为野生精灵 |
| 背包生效 | ✅（队伍任意位置即可） |
| 叠加组 | `run_rate` |
| 首道馆前 | ✅ 允许 |
| 流派绑定 | 融合线《理念之争》首次进化奖励 |
| 设计意图 | 强化"速度机动"定位；等级相等或低于时无保证，维持探索张力 |
| 风险备注 | 等级判断必须用持有者等级，不能用队伍平均等级，否则早期队均等级膨胀导致全程 100% 逃跑 |

---

#### `sustain_a` — 续航

| 项目 | 值 |
|------|----|
| 类型 | adventure（冒险） |
| 效果 | 持有者在**后备/背包**中，每 tick 回复 maxHp 的 **2%** |
| 触发条件 | 持有者处于非上场状态，每探索 tick 触发 |
| 上限 | 不超过 maxHp |
| 建议区间 | 1.5% ~ 3.0% / tick |
| 当前取值 | 2% / tick |
| 背包生效 | ✅ |
| 叠加组 | `reserve_regen` |
| 首道馆前 | ✅ 允许 |
| 流派绑定 | 吞噬线《理念之争》首次进化奖励 |
| 设计意图 | 强化"续航稳定"定位；减少外出补给次数，低调但累积收益可感知 |
| 风险备注 | tick 粒度由主程侧定义，建议 = 每次遭遇结算后；若 tick 太密集（如每步）需下调至 0.5% |

---

### S 级特性

> **首道馆前禁止发放。** 以下条目仅为规格预埋，投放时机由主程侧在道馆解锁事件后切换 `preGymAllowSTier = true`。

#### `double_yield_s` — 回赠

| 项目 | 值 |
|------|----|
| 类型 | adventure（冒险） |
| 效果 | 队伍任意位置有该持有者，野外遭遇奖励（金币、掉落素材）**×2.0** |
| 触发条件 | 队伍内存在持有者（背包即可），战斗/遭遇结算时生效 |
| 建议区间 | ×1.5 ~ ×2.5 |
| 当前取值 | ×2.0 |
| 背包生效 | ✅ |
| 叠加组 | `wild_reward` |
| 首道馆前 | ❌ 禁止 |
| 风险备注 | 不影响商店交易与任务固定奖励；若叠加经济加成技能需联合审查通货膨胀风险 |

---

#### `unyielding_s` — 不屈

| 项目 | 值 |
|------|----|
| 类型 | battle（战斗） |
| 效果 | 每场战斗**首次**受到致命伤害时，HP 保留 1 而非 KO |
| 触发条件 | 当次伤害使 HP 从 >0 变为 ≤0，且本场 `fatalBlockUsed = false` |
| 重置时机 | 每场战斗开始时 `fatalBlockUsed` 重置 |
| 保留 HP | 1（固定） |
| 背包生效 | ❌（持有者须出战） |
| 叠加组 | `fatal_block` |
| 首道馆前 | ❌ 禁止 |
| 风险备注 | 道馆战/剧情 Boss 中需确认 `fatalBlockUsed` 每场战斗独立重置，防止同一 Boss 分段战斗被无限触发 |

## 三、仁慈技能（通用，非特性）

| 项目 | 值 |
|------|----|
| 技能类型 | 无属性技能（通用） |
| 基础威力 | 60 |
| 伤害计算 | 走标准伤害公式，无 STAB 加成 |
| 留 1 HP 条件 | `battleContext.isWild === true` |
| 留 1 HP 逻辑 | 计算伤害 ≥ 目标当前 HP 时，将目标 HP 设为 1（不 KO）|
| 例外 | 若目标 HP 已是 1，正常造成最小 1 点伤害（允许 KO） |
| 训练师战 / 道馆战 / Boss | `isWild = false`，留血无效，正常 KO |
| 发放路径 | 融合线与吞噬线《理念之争》结束均获得 |

### 模拟局 1：融合线（一往无前 + 仁慈）

**进化后神兽幼体属性假设（基础 Lv.6 进化后约等于 Lv.7 档位）：**

| 属性 | 进化前（Lv.6 示例值） | 进化后（×倍率） |
|------|------------------|--------------|
| 攻击 | 18 | 18 × 1.08 ≈ **19** |
| 防御 | 14 | 14 × 1.06 ≈ **15** |
| 最大 HP | 40 | 40 × 1.08 ≈ **43** |
| 速度 | 16 | 16 × 1.22 ≈ **20** |

**vs 花冠大道 Lv.2 野怪（stonehorn，DEF≈9，ATK≈10）：**

```
对手攻击方：offenseBase = 10 × 0.56 + 2 × 1.9 = 5.6 + 3.8 = 9.4
我方防御：defenseBase × 0.52 + level × 1.9 = 15 × 0.52 + 7 × 1.9 = 7.8 + 13.3 = 21.1
rawDamage = max(1, floor((9.4 - 21.1) × 1.0 × 1.0)) = max(1, -12) = 1
```
→ **对手每次攻击仅 1 点伤害，43 HP 可承受 42 次攻击** ✅ 碾压感成立。

**我方攻击对方（逃跑测试）：**
持有 `fearless_stride_a`，野怪 Lv.2 < 我方 Lv.7，**逃跑成功率 100%**。
仁慈攻击对方 HP 到 1 后捕捉，野生战留血逻辑生效。 ✅

**vs 馆主 Lv.10 bloomantis（草系，ATK≈24，DEF≈20）：**

```
对手攻击我方（草 vs 无克制）：
offenseBase = 24 × 0.56 + 10 × 1.9 = 13.4 + 19 = 32.4
我方防御 = 15 × 0.52 + 7 × 1.9 = 7.8 + 13.3 = 21.1
rawDamage = max(1, floor((32.4 - 21.1) × 1.0)) = 11
```
→ 43 HP 可承受约 **3~4 次**普通攻击，需适时使用药水。

若玩家无克制属性（火/飞行/虫）：进攻效率低，需多回合消耗，约需 2~3 瓶药水。
若带火系技能（无特性加成）：`blaze_boost_b` 提升约 10%，约减少 1 回合。
**结论：融合线不带克制属性打草系道馆有压力，符合"需策略才能通关"设计目标。** ✅

---

### 模拟局 2：吞噬线（续航 + 仁慈）

**进化后神兽幼体属性（同 Lv.6 基底）：**

| 属性 | 进化前 | 进化后（×倍率） |
|------|--------|--------------|
| 攻击 | 18 | 18 × 1.11 ≈ **20** |
| 防御 | 14 | 14 × 1.09 ≈ **15** |
| 最大 HP | 40 | 40 × 1.13 ≈ **45** |
| 速度 | 16 | 16 × 1.04 ≈ **17** |

**vs 花冠大道 Lv.2 野怪（同上）：**
防御与融合线相近，**受到伤害同样约 1 点**，45 HP 可承受 44 次。 ✅

**续航特性收益估算（每 tick 回 2%）：**
后备精灵每次探索遭遇后回 45 × 2% ≈ **0.9 HP**（约 1 HP/遭遇）。
连续 5 次探索约回复 4~5 HP，相当于约半瓶药水的累积价值。
对短时间节约消耗感知不明显，但多场景连续探索（10+ 次）后体感明显。 ✅

**vs 馆主 Lv.10 bloomantis：**

```
我方攻击（ATK=20）：
offenseBase = 20 × 0.56 + 7 × 1.9 = 11.2 + 13.3 = 24.5
对手防御 = 20 × 0.52 + 10 × 1.9 = 10.4 + 19 = 29.4
rawDamage = max(1, floor((24.5 - 29.4) × 1.0)) = 1
```
→ 无克制属性时我方攻击 bloomantis 每次仅 1 点，**强制需要草系克制技能**。 ✅

**结论：吞噬线进攻厚度略高（ATK +11% vs 融合 +8%），HP 更厚（45 vs 43），速度较低（17 vs 20）。与融合线形成非对称但等价的体验差异。** ✅

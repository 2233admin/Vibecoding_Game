# 特性分级与《理念之争》数值策划任务单

## 1. 目标
- 建立 `B / A / S` 特性分级体系，避免“特殊进化=必得顶级特性”导致稀缺性崩坏。
- 将《理念之争》主线奖励做成“可感知成长 + 可控平衡”的中期教学闭环。
- 保持 Demo 节奏：神兽幼体 -> 理念之争 -> 引导挑战道馆。

## 2. 适用范围
- 当前比赛 Demo（新手村、花冠大道、蒙德草原、首个道馆前后）。
- 本文用于数值策划和系统实现对齐，不涉及美术产出规范。

## 3. 核心规则（已确认）
1. 平衡目标：融合流派与吞噬流派“非对称但等价”。
2. 特性稀缺：仅稀有精灵、传说精灵、神兽可拥有特性。
3. 技能 `仁慈`：只对野生精灵生效。
4. 《理念之争》结束后，NPC 必须明确引导玩家去道馆。

## 4. 特性分级定义

### B 级（常见强化，低风险）
- 定位：基础增益，提升手感但不改变玩法结构。
- 典型强度：单项增益 `5%~12%`。
- 示例：
  - `火势增幅`：火系技能伤害 +10%。
  - `奋勇`：当前血量越低，造成伤害越高（上限 +15%）。

### A 级（构筑关键，明显改变冒险/战斗决策）
- 定位：流派辨识度核心，属于“有价值但不过分破坏平衡”的能力。
- 典型强度：可改变战斗/探索策略，但有边界条件。
- 示例：
  - `一往无前`：对等级低于自身的野生精灵，逃跑成功率 100%。
  - `续航`：拥有该特性的精灵在背包/后备中按固定速率回复 HP。

### S 级（稀缺上限，后期目标）
- 定位：强力且稀有，不进入首道馆前主线常规发放。
- 示例：
  - `回赠`：野外探索奖励 x2（金币、素材、掉落）。
  - `不屈`：战斗中免疫一次致命伤害并保留 1 HP。

## 5. 稀缺与投放规则
1. 普通精灵：无特性。
2. 稀有/传说/神兽：可配置 B/A/S 特性。
3. 首道馆前默认不投放 S 级特性（避免前期经济和战斗被冲垮）。
4. 《理念之争》首次特殊进化建议发放 A 级特性（流派绑定）。

## 6. 《理念之争》落地方案（v1）

### 融合流派
- 叙事奖励：获得白翼素材精灵并完成一次融合教学。
- 进化结果：神兽幼体获得白翼形态。
- 数值方向：
  - 攻击：小幅提升（建议 +8%）
  - 防御：小幅提升（建议 +6%）
  - 最大 HP：小幅提升（建议 +8%）
  - 速度：明显提升（建议 +20%~25%）
- 新特性（A）：`一往无前`
- 新技能：`仁慈`

### 吞噬流派
- 叙事奖励：获得残破圣剑并完成一次吞噬教学。
- 进化结果：神兽幼体武装化（叼剑或局部武器化）。
- 数值方向：
  - 攻击：中幅提升（建议 +10%~12%）
  - 防御：小幅提升（建议 +8%~10%）
  - 最大 HP：中幅提升（建议 +12%~15%）
  - 速度：小幅提升（建议 +3%~6%）
- 新特性（A）：`续航`
- 新技能：`仁慈`

### 通用技能：仁慈（无属性）
- 基础威力：中等（建议 55~65）。
- 生效边界：
  - 仅对野生精灵触发“留 1 HP”。
  - 对训练师战、道馆战、剧情 Boss 不触发留血效果。
- 设计目标：提升捕捉体验与图鉴推进效率。

## 7. 冒险特性生效规则（背包生效）
1. 特性持有者不必出战，放在背包/后备即可生效。
2. 生效边界建议：
  - 同类效果不叠加，取最高值。
  - 若后续出现冲突，按优先级 `S > A > B`。
3. 初版只开放少量冒险特性，避免规则复杂化。

## 9. 主线引导文案要求
《理念之争》战斗结算后，NPC 必须出现以下信息：
1. 神兽进化只是起点，真正的机遇在道馆试炼中。
2. 目标栏更新为“前往道馆”。
3. 给出道馆入口方向提示（防止迷路）。

## 10. 数值策划执行清单
1. 完成 B/A/S 特性表与说明。
2. 给融合/吞噬两线填写首版倍率并标注可调区间。
3. 落地 `仁慈` 的野生战限定逻辑参数。
4. 配置《理念之争》奖励产出（素材、技能、特性）。
5. 增加至少 1 轮首道馆前实测记录（融合线/吞噬线各一局）。

## 11. 验收标准
1. 玩家在《理念之争》中必定体验一次特殊进化。
2. 玩家能明确感知流派差异（速度机动 vs 续航稳定）。
3. `仁慈` 在野生战可用，在训练师战无效。
4. 主线完成后玩家能被稳定引导至道馆。
5. 首道馆前整体难度不被特性系统击穿（无明显“无脑碾压”）。

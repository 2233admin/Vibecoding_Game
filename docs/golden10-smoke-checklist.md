# 黄金10分钟烟测清单（2026-03-28）

## 0. 测试前置
- 浏览器强刷：`Ctrl + F5`
- 确认资源版本：`index.html` 已是 `v=20260328h`
- 建议先在“选择存档”里用空槽位开新档，避免旧档干扰

## 1. 开局序章
1. 点击“开始游戏”
2. 预期：立即弹出 `神兽序章（黄金10分钟开场）`
3. 选择“开始序章”或“直接选择”均可继续

## 2. 流派选择与镜前登记
1. 预期：出现 `古神试炼：你支持哪一方？`
2. 选择融合或吞噬后，进入镜前身份登记
3. 预期：依次出现
   - `梦醒镜前：登记你的训练家身份`
   - `镜前立绘选择`
4. 完成立绘选择后，预期关闭选择框并提示去找教授

## 3. 领初始精灵与教学战
1. 与教授雪松交互，选择初始精灵
2. 预期：进入教学战（教程保护开启）
3. 结束后预期：`storyStage` 进入 1 阶段

## 4. 出村神兽幼体教学捕捉
1. 从星辉城右侧出口离开（`>`）
2. 预期：触发流派对应神兽幼体遭遇
3. 预期：战斗中仅允许投球（技能/道具/逃跑/切换禁用）
4. 预期：捕捉必定成功，战后提示契约完成

## 5. 回归判定（通过标准）
- 新档“开始游戏”必触发序章入口
- 序章流程可完整走通，不会卡死或重复弹窗
- 出村可触发“神兽幼体必遇必得”捕捉教学

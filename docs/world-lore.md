# World Lore

## 核心神话

上古时代，两尊原初神兽因进化理念分歧而爆发大战。

- 吞噬之道：力量来自掠夺与独占。
- 融合之道：力量来自连接与共生。

大战之后，两尊神兽并未被证实死亡。后世更普遍的说法是：它们在神战中失去完整神性，陷入沉睡。它们残留的力量分散为数道古老权能，沉入各地遗迹、道馆传承与守护仪式之中。

吞噬与融合都不是凭空创造新神，而是让沉睡古神重新完整化的不同复苏手段。

## 权能设定

权能是神兽完整神性的碎片，不是机械部件，也不是普通素材。

- 每一道权能都对应古神的一部分本质。
- 道馆与守护者负责保管、筛选与试炼，而不是随意发放。
- 玩家获得权能，不代表“拼装神兽”，而代表幼体与失落神性重新建立共鸣。

## 道馆与试炼规则

本项目最重要的世界观规则：

**徽章证明的是训练师的实力，权能试炼验证的却是继承古神之力的资格。**

这条规则用于解释以下问题：

1. 为什么很多训练师可以挑战道馆，却无法触及古神之力。
2. 为什么即使是冠军级训练师，也不一定能让精灵走向神兽复苏路线。
3. 为什么主角必须同时满足“实力、共鸣、被认可”三项条件。

标准流程应为：

1. 玩家先完成正常道馆战，证明自己具备训练师实力。
2. 若队伍中携带对应神兽幼体，且满足共鸣条件，馆主才会开启隐藏的权能试炼。
3. 通过试炼后，幼体吸纳或共鸣该权能，进入下一阶段成长。

## 两派与真正反派

吞噬派与融合派都不是简单的善恶对立，它们是对于“古神应如何复苏”的理念之争。

- 吞噬派认为，失落神性必须通过夺取与归一重聚。
- 融合派认为，失落神性必须通过连接与共鸣重现。

真正的终局反派，不应只是两派中的任意一方，而是那个主动激化两派矛盾、利用古神复苏进程谋取自身目的的人。

## 与自然进化派的关系

自然进化派可以作为第三条世界观暗线存在。

- 吞噬派强调主动夺取。
- 融合派强调主动连接。
- 自然进化派强调生命应顺其本性成长。

后续主线可将自然进化派作为伏笔，用来反衬前两派的极端化风险，并为幕后反派提供更复杂的思想来源。

## 进化体系核心原则

1. 普通进化是世界主流。
   - 绝大多数精灵保留 2 段或 3 段正常进化链。
   - 吞噬与融合都需要训练家、素材、仪式或试炼参与，不是生态中的普遍现象。
2. 对战生态上，普通流派与特殊进化流派应长期共存。
   - 在高水平对战环境里，普通流派与特殊进化流派都应具备稳定出场空间。
   - 目标不是让特殊进化全面淘汰普通进化，而是形成大致各占半壁的生态。
3. 特殊进化更强调素材策略与操作。
   - 吞噬与融合的强点主要来自训练家对素材、时机、队伍构筑与战术执行的掌控。
   - 特殊进化不应只靠全面更高的数值取胜。
   - 前中期融合靠少量素材做策略，终局融合才允许完整神兽化。
   - 双体融合是绝对主流，也是现代训练家可稳定掌握的标准形态。
   - 三体及以上融合属于传说层级个例，不作为常规培养与对战体系的主流规则。
   - 同阶融合是绝对主流，约占融合案例的 95%。
   - 低阶个体更容易进入融合态；高阶个体更难完美融合。
   - 个体境界越高，可承受的融合数量越少；三阶段精灵通常最多只能容纳 2 个融合个体，神兽是例外。
4. 三条道路都能通向强者。
   - 自然进化、吞噬进化、融合进化都能培养出顶级战力。
   - 它们的区别在于代价、门槛、养成方式与最终上限不同。
5. 神兽级特殊进化必须通过试炼与引导完成。
   - 普通个体只可能在极端条件下发生不完整异变。
   - 这类异变个体的理论上限最高只到准神兽级别，不能自然走到完整古神复苏。

## 特殊进化与繁衍原则

1. 境界越强的个体，生育越难。
   - 这是普通个体、特殊进化个体与神兽级存在共同遵守的底层规律。
2. 吞噬进化的后代通常回归正常个体。
   - 后代不会直接继承完整吞噬形态。
   - 只会以较低概率继承“吞噬痕”或少量特殊天赋。
3. 融合进化默认是长期存在的稳定状态，但不是天然永久。
   - 正常融合个体可以长时间维持进化形态，并通过持续训练逐步提升稳定度。
   - 只有完成稳定共鸣并达到“完美融化”的个体，才会被视为新的完整生命形态。
4. 融合个体的繁育资格取决于稳定程度。
   - 未完成完美融化的融合个体，不视为稳定繁殖模板。
   - 完成完美融化后，才可被世界视为正常个体并具备常规繁育资格。
5. 神兽级个体不参与常规繁殖体系。
   - 它们属于复苏、继承与试炼体系，而非普通种群延续体系。
